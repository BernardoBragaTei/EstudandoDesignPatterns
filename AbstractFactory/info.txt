#Padão Abstract Factory
- Abstração de fábricas de objetos
- Serve para abstrair a instanciação de famílias de produtos (conjuntos de classes que trabalham juntas)
- Ex.: Abstração de componentes de interface de usuário para diferentes plataformas (janelas, barras de rolagem, botões, etc)

Estudo baseado na aula do Macoratti
